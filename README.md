# Iron Lady – AI & Technology Intern Assignment

This repository contains solutions for the Iron Lady AI & Technology Intern assignment.

---

## Task 1: AI-based Customer Interaction Solution
An AI-powered guidance assistant designed to support Iron Lady’s learners by:
- Explaining programs and offerings
- Guiding users through leadership journeys
- Improving engagement through conversational AI

**Tech Stack:**
- Frontend: React
- Backend: Spring Boot
- AI: LLM-based API

Folder: `task-1-ai-assistant`

---

## Task 2: Internal Business Automation – Lead Management
A lead management system built to support Iron Lady’s internal operations:
- Create, view, update, and delete leads
- Track program interest and lead status
- Reduce manual follow-ups and improve efficiency

**Tech Stack:**
- Frontend: React
- Backend: Spring Boot
- Database: H2 (in-memory)

Folder: `task-2-lead-management`

---
